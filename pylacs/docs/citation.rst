Citation
========

If you use pylacs in published research, please cite both the software and the methodological foundation.

Software citation
-----------------

Cite the pylacs version used in your analysis (reported as |version| in these docs).
If you mint a DOI (e.g., Zenodo), cite the DOI-associated record.

Methodological background
-------------------------

Include appropriate citations for the LACS formulation and the random-coil reference models.
Add entries to ``docs/refs.bib`` and include them using:

.. bibliography:: refs.bib
   :all:
